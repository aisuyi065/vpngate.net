# Hysteria 2 Native Mode Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a `hy2-native` runtime mode that manages a local Hysteria 2 server on no-TUN LXC hosts, exposes backend APIs for status/configuration, and provides a one-shot installer plus dashboard support.

**Architecture:** Keep the existing FastAPI/Vue app, but add a new Hysteria-specific service manager and API surface alongside the legacy OpenVPN path. The installer will call the official Hysteria installer and let this project own the generated config, runtime metadata, and UI.

**Tech Stack:** Python 3.12, FastAPI, Pydantic, Vue 3 + TypeScript, systemd, shell installer, official Hysteria 2 service.
